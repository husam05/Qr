## Print-Ready QR Codes

### For Business Cards:
```
Size: 400x400px minimum
Format: PNG (high quality)
Margin: Add white border
Test: Scan before printing!
```

### For Posters/Flyers:
```
Size: 800x800px or larger
Format: PNG or SVG
Colors: High contrast
Placement: Eye level, good lighting
```

### For Table Tents (Restaurants):
```
Size: 400x400px
Format: PNG
Colors: Match restaurant branding
Text: Add "Scan for Menu" label
```
